Methods of String and Array.

String Methods are as follows

1.Substring

2.Substr

3.Slice.

1. str.slice(start [, end])
This method is use to slice any strings according to their index. let str = "Aakash";
console.log( str.slice(0, 3) ); // 'kash', the substring from 0 to 5 (not including 3)

If we dont provide 2nd args, then slicing will from 1st index to last index of the string:
console.log( str.slice(2) );

2. str.substring(start, [end])
Returns the part of the string between start and end.
This is almost the same as slice, but it allows start to be greater than end.


3. str.substr(start [, length])
Returns the part of the string from start, with the given length.
In contrast with the previous methods, this one allows us to specify the length instead of the ending position:
console.log( str.substr(2, 4) ); // 'ka', from the 2nd position get 4 characters
// The first argument may be negative, to count from the end:

The call to arr.sort() sorts the array in place, changing its element order.

It also returns the sorted array, but the returned value is usually ignored, as arr itself is modified. let arr4 = [ 92, 29, 119,220 ]; // the method reorders the content of arr

arr.sort(); alert( arr ); [119, 220, 29, 92]
Did you notice anything strange in the outcome? The order became[119, 220, 29, 92]. Incorrect. But why? The items are sorted as strings by default. Literally, all elements are converted to strings for comparisons. For strings, lexicographic ordering is applied and indeed "2" > "15". To use our own sorting order, we need to supply a function as the argument of arr.sort(). The function should compare two arbitrary values and return:

function compareNumeric(a, b) {

if (a > b) return 1;

if (a == b) return 0;

if (a < b) return -1;

}
